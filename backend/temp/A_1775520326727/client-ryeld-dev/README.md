# client-ryeld
## Repositorio privado del cliente del sistema web para la empresa Ryeld

### 🚀 Correr el proyecto: 
```bash
pnpm install
pnpm run dev
```

### 🔧 Rama sin Backend (dev-sinbackend)

Esta rama está configurada para trabajar **sin necesidad de backend**. Puedes hacer modificaciones visuales y probar toda la interfaz sin depender del servidor.
